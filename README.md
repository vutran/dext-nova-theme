# dext-nova-theme

> [Nova](http://www.trevordmiller.com/nova/) theme for [Dext](https://github.com/vutran/dext).

![](screenshot.png?raw=true)

## Install

```bash
# install the theme
$ dpm install dext-nova-theme

# sets as the new theme
$ dpm theme dext-nova-theme
```
